简介：将文件复制到其他路径的工具
### 一、包的安装和引用
```
// 安装 cp-cli 命令行工具
npm install --global cp-cli
```
### 二、cp-cli的使用

```
// 终端直接执行comman命令
// cp-cli 文件（夹） 去往的地址,如：将根路径下的a文件夹下的a.txt移到根目录b文件夹下，取名b.txt，没有文件则新建
cp-cli a/a.txt b/b.txt
```
